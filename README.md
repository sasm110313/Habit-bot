# 🎯 ربات پیشرفته عادت‌سازی | Habit Tracker Bot v2.0

ربات تلگرام پیشرفته برای پیگیری ۳ عادت اصلی با سه سطح، یادآوری هوشمند دوره آموزشی، و آمار دقیق.

---

## ✨ امکانات

### 🕌💪🌙 سه عادت ثابت — هر کدام با ۳ سطح

| عادت | 🟢 لقمه کوچک | 🟡 لقمه ویژه | 🔴 لقمه اضطراری |
|------|-------------|-------------|-----------------|
| تمرکز در نماز | نماز کامل ۱۵ دقیقه | حداقل تمرکز ۵ دقیقه | نیت + حضور ۱ دقیقه |
| خواب منظم | قبل ۲۳ + بیدار ۶ | قبل ۲۴ + بیدار ۷ | حداقل ۶ ساعت |
| ورزش | ۱۵ دقیقه کامل | ۵ دقیقه سبک | ۱ دقیقه حرکت |

### ⏰ یادآوری‌های خودکار

- **📚 دوره آموزشی**: ۵ بار در روز (۸:۰۰ | ۱۲:۳۰ | ۱۷:۰۰ | ۲۰:۳۰ | ۲۲:۳۰)
- **📋 عادت‌ها**: ۴ بار در روز (۷:۳۰ | ۱۳:۰۰ | ۱۸:۳۰ | ۲۱:۰۰)
- **🌙 خلاصه شبانه**: ۲۲:۴۵

### 📊 آمار و پیشرفت

- آمار هفتگی با نمودار
- آمار ماهانه با توزیع سطوح
- 🔥 استریک با رکورد بهترین
- پیام‌های انگیزشی متنوع

## 🚀 نصب (یک دستور!)

```bash
git clone https://github.com/sasm110313/Habit-bot.git
cd Habit-bot
sudo bash install.sh "YOUR_BOT_TOKEN"
```

## 🔄 بروزرسانی

```bash
cd Habit-bot
git pull
sudo bash install.sh
```

> نکته: موقع آپدیت نیازی به توکن نیست — خودش از نسخه قبلی میخونه.

---

## 📱 نحوه استفاده

1. `/start` بزن
2. روی **📋 وضعیت امروز** کلیک کن
3. روی هر عادت کلیک کن → سطح رو انتخاب کن
4. تمام! ✅

---

## 🛠 مدیریت سرور

```bash
sudo systemctl status habit-bot      # وضعیت
sudo systemctl restart habit-bot     # ریستارت
sudo journalctl -u habit-bot -f      # لاگ زنده
sudo systemctl stop habit-bot        # توقف
```
