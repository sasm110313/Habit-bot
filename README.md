# 🎯 ربات عادت‌سازی v3.0 | Habit Tracker Bot

ربات پیشرفته تلگرام برای عادت‌سازی با گیمیفیکیشن، یادآوری هوشمند، و تحلیل شبانه.

---

## ✨ امکانات

### 🕌💪🌙 سه عادت — سه سطح

| عادت | 🟢 لقمه کوچک (30 XP) | 🟡 لقمه ویژه (20 XP) | 🔴 اضطراری (10 XP) |
|------|-----|-----|-----|
| تمرکز در نماز | ۱۵ دقیقه | ۵ دقیقه | ۱ دقیقه |
| خواب منظم | قبل ۲۳ + بیدار ۶ | قبل ۲۴ + بیدار ۷ | ۶ ساعت |
| ورزش | ۱۵ دقیقه | ۵ دقیقه | ۱ دقیقه |

### 🎮 گیمیفیکیشن
- **XP** — هر عادت، دوره، تحلیل امتیاز داره
- **۱۰ لول** — تازه‌کار → مبتدی → تلاشگر → ... → عادت‌ساز 🏆
- **دستاوردها** — ۲۰+ مدال قابل آنلاک
- **استریک** — با رکوردشکنی!
- **بونوس** — روز کامل +50 XP!

### ⏰ یادآوری هوشمند
- فقط وقتی **انجام نداده** باشی!
- **صبح**: آیه قرآن + انگیزه (۶:۳۰)
- **دوره**: ۵ بار/روز (۸ | ۱۲:۳۰ | ۱۷ | ۲۰:۳۰ | ۲۲:۳۰)
- **عادت**: ۶ بار/روز (۷:۳۰ | ۱۰ | ۱۳ | ۱۶ | ۱۸:۳۰ | ۲۱)
- **تحلیل**: ۲۲:۰۰
- **خلاصه شبانه**: ۲۲:۴۵

### 📚 دوره آموزشی
- ۹ چهله (۳۶۰ جلسه)
- پیگیری شماره جلسه
- استریک جداگانه

### 📝 تحلیل شبانه
- ژورنال آزاد هر شب
- +15 XP برای نوشتن

## 🚀 نصب

```bash
git clone https://github.com/sasm110313/Habit-bot.git
cd Habit-bot
sudo bash install.sh "YOUR_BOT_TOKEN"
```

## 🔄 بروزرسانی

```bash
cd Habit-bot
git pull
sudo bash install.sh
```
